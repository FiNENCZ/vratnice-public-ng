Struktura CSV souboru

  • typVozidla: Typ vozidla, povolené hodnoty jsou osobní, dodávka, nákladní, speciální. Můžete zadat více typů oddělených vertikální čárou (|).
  • rzVozidla: RZ vozidla, můžete zadat více hodnot oddělených vertikální čárou (|).

Názorný příklad CSV soubouru:

        typVozidla,rzVozidla
        osobní|nákladní,6asdsad|65464

CSV File Structure

  • typVozidla: Vehicle type, allowed values are osobní (personal), dodávka (van), nákladní (truck), specialní (special). You can enter multiple types separated by a vertical bar (|).
  • rzVozidla: Vehicle registration number, you can enter multiple values separated by a vertical bar (|).

Example CSV File:

        vehicleTypes,vehicleLicensePlates
        personal|van,6asdsad|65464
